# POS-Foodi

A Spring Boot-based Point of Sale system for food businesses.

## Features
- User Management
- Order Processing
- Inventory Tracking
- REST APIs

## Tech Stack
- Java
- Spring Boot
- Maven

## Getting Started
1. Clone the repo
2. Run `mvn spring-boot:run`
3. Access via `http://localhost:8080`
